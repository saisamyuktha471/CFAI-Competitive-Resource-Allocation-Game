Competitive Resource Allocation Game

Artificial Intelligence Mini Project

Welcome to the AI Strategy Arena

This project demonstrates intelligent decision-making using Minimax Algorithm, Alpha-Beta Pruning, Bayesian Prediction, Heuristic Evaluation, and Constraint Satisfaction Problems.

Minimax Algorithm

Evaluates future game states to select the optimal move.

Alpha-Beta Pruning

Reduces unnecessary computations and improves efficiency.

Bayesian Prediction

Predicts opponent actions using probability distributions.

CSP Validation

Ensures all resource allocations satisfy constraints.

Game Rules

  • Each player starts with 20 resources.
  • The game consists of 5 turns.
  • Resources allocated become score points.
  • The player with the highest score wins.
